1. Instale o React Navigation e dependências necessárias:
Primeiro, você precisa instalar o react-navigation e as dependências necessárias. Execute os seguintes comandos no seu terminal dentro do diretório do projeto:

1 - npm install @react-navigation/native
2 - npm install @react-navigation/stack
3 - npm install react-native-screens react-native-safe-area-contexto


Após a instalação, execute o seguinte comando para vincular as dependências nativas no seu projeto:
